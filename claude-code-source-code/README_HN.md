## लापता मॉड्यूल सूचना (108 मॉड्यूल)

> **यह स्रोत अधूरा है।** `feature()`-गेटेड शाखाओं द्वारा संदर्भित 108 मॉड्यूल एनपीएम पैकेज में **शामिल नहीं** हैं।

वे एन्थ्रोपिक के आंतरिक मोनोरेपो में मौजूद हैं। बुन (Bun) का `feature()` संकलन-समय (compile-time) पर काम करता है:
- एंथ्रोपिक के आंतरिक निर्माण में `true` लौटाता है → कोड बंडल में रहता है
- प्रकाशित निर्माण में `false` लौटाता है → डेड-कोड को हटा दिया जाता है (DCE)

## द एजेंट पैटर्न (The Agent Pattern)

```text
                    मुख्य लूप
                    =========

    उपयोगकर्ता --> messages[] --> Claude API --> प्रतिक्रिया (response)
                                           |
                                 क्या stop_reason == "tool_use" है?
                                /                          \
                              हाँ                           नहीं
                               |                             |
                         उपकरण (tools) चलाएं             टेक्स्ट लौटाएं
                         tool_result जोड़ें
                         वापस लूप करें --------------> messages[]
```

यह न्यूनतम एजेंट लूप है। क्लॉड कोड इस लूप को उत्पादन-स्तरीय हार्नेस के साथ लपेटता है: अनुमतियाँ (permissions), स्ट्रीमिंग, समवर्ती (concurrency), उप-एजेंट (sub-agents), दृढ़ता (persistence) और एमसीपी (MCP)।
